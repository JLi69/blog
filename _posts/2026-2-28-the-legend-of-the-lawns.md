It's been a couple months since I've lasted posted on this blog. So as a few
quick updates on general things that I've been working on, I've finished my
first semester of university which feels really nice, I had a pretty relaxing
winter break in which I was able to get quite a bit done for my current project
(*The Legend of the Lawns*, or LotL, which I will talk about in this post), and
I've also started my second semester of university which has mostly been going
well but of course with school going on, it's a little more difficult to work
on my own personal projects which is kind of unfortunate.

I have not written about this project on this blog until now so I should probably
introduce it and give some background on what's been done so far. I started working 
on a game with one of my friends (gldeA) for [Birb Games](https://github.com/Birb-Games)
in June of 2025 - it was the summer so I saw a lot of people mowing lawns around
my neighborhood so the idea of a game about mowing lawns was kind of stuck in
my head and that's the idea we decided to go with. I thought that the project
would only last the summer but it ended up taking some extra time and I then
got a little distracted by college and also was working on some other small
projects (like developing Voxelworld v0.5.0) so LotL development went on hold.
I did eventually start getting more done around winter break and during the
semester I also recruited another person to help out with development 
([Bridget Wu](https://github.com/BridgetWu)). Of course, with the start of the
new semester development once again slowed down and my friend gldeA decided to
step back from Birb Games since he was busy with other things now (which is
totally fine, I wish him luck on all of his future endeavors). I have continued
work on adding new features to LotL, the scope for the project has grown quite
a bit but I do feel that I have a good part of the game done. I have also
made a gameplay video [on YouTube](https://www.youtube.com/watch?v=EXcu8_R3Tdw)
if anyone wants to check it out.

## Alright, so what's next?

Currently I'm working on adding a store in which the player can buy upgrades
and items. I've already implemented a number of items but I still have more to
add so that's probably going to be my main personal project for the next month
or so.

Once I get that done, I only really need to just start 'putting together' the
game's features to make a cohesive final product.

The major things that I need to get done are:
 - Designing all of the various lawns
 - Adding some story details (I don't intend on having a complicated story for
 the game but I still want to do some cool things - which I'm not going to
 disclose here, no spoilers!)
 - Polishing up the game and hopefully fixing any bugs I find and just making
 it a good experience - I also want to ensure the game is somewhat balanced
 - Just adding some charm via NPCs and small details.

There's still quite a lot to do and development might be kind of slow, but I am
setting a release date to be around summer of 2026 (around the time when everyone
seems to be mowing their lawn :P) so hopefully we're able to make it. 
College is busy though.

I don't really intend on growing the scope any further, I do want to complete the
project within a few months and get it released for people to check out. I really
do like how it's coming together though - I'm thinking LotL can kind of like
a "bullet hell" style game and one of the things that I want is that as the
player progresses, things just get more and more chaotic when they're mowing the
lawn. That sounds pretty cool in my opinion.

I'll be honest, as much fun as it is seeing this game slowly become more completed
by the day, I am getting a little tired working on it, it is kind of rough
having to work on the same project for the better part of a year and likely for
a few months more. I probably should take a break after LotL releases and then
move onto some other projects. Maybe work a bit on Voxelworld v0.6.0, or just
start a new project - I do have quite a number of ideas for new games that I
want to make. We'll see. Oh yeah, I guess as a quick tangent about Voxelworld,
I do want to work on v0.6.0 and add animals/monsters to the game but that's going 
to be a whole giant project that's probably going to take a while and it's already
for a project that I've worked on for quite a while and I kind of want to work
on something new/original, so Voxelworld development might be stalled for a while
(it already is stalled but probably for a while after the release of LotL as well).
Anyway, hopefully I can continue getting more features added bit by bit to LotL
and get this project done on time.

Anyway, that's it for now, have a nice day!
